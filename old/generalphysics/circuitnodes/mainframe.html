elements:

This page will model any passive circuit with resistors and batteries (including bridges). In the elements form field, enter batteries like:

{V, Voltage value, low node, high node}

and enter resistors like:

{R, Resistance value, low node, high node}

Note that elements need to be separated by commas. You may also add in hard returns after the commas.

After drawing your circuit, number the nodes (connections between any elements) and low node will be the low number and high node will be the high number for a given element. For batteries, the voltage value should be >0 if the low node is connected to the +.

All currents are assumed to flow from low node to high node. Ground is assumed at the highest node number.

The page to the right will display first a table of currents for each element, then a table of voltage levels at each node and finally a list of the equations used.

The default example is a cube with the battery hooked up from corner to corner (labeled 1 and 8).

Finally, a graph is shown where the nodes are on a circle and all elements are shown. The batteries are shown in red and the resistors in blue. This uses the Combinatorica package in Mathematica.
